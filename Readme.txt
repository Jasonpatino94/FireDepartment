Explanation:
This is a website to show off my Fire Department. It tells you about the fire department, what we do as firefighters, shows you pictures and videos from drills or calls
shows you the members of the fire department and also lets you donate to us. Also for locals, it lets you download some applications to actually be part of the team.
If you log in as an admin, you can do CRUD on firefighter and or Certificates.

How to run the app:
You have to run The-Fire-Department on the server and run both FirefighterRest and FirefighterSecurity as a springboot project.

Usage:
One you have everything set up and running you can click throughout the web site to get some knowledge of my firedepartment.
you can look at videos, find pictures, donate or apply to become a firefighter
If you have access to the admin account you are able to log in and then use the other pages to create, update, delete a firefighter/certificate or just see all of them
